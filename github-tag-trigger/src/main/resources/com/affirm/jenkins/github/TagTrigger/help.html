Listens for push events from GitHub, and triggers a new build when a tag is updated or created.

To use this trigger,

  1. Under "Git Repositories > Advanced", use +refs/tags/*:refs/remotes/origin/tags/*.
  2. Under "Branches to build > Branch Specifier", use either ${GIT_REF} (for the full reference) or ${TAG_TO_USE} (for just the tag name).
